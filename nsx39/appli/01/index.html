ポケット・ミクと接続

ポケット・ミクを接続しブラウザを再起動してください。

ブラウザの設定

本アプリケーションを利用するためには以下の設定が必要です。
  • chrome://flags/#enable-web-midi を有効化
    1. アドレスバーに chrome://flags/#enable-web-midi を入力
    2. Web MIDI API を有効にする を有効化
    3. 画面最下段の をクリックしてブラウザを再起動

ポケットミクへ再接続中です。

このダイアログは再接続が完了すると自動的に消えます。

ダイアログが消えない場合、お手数ですがポケット・ミクの再起動後にブラウザの再起動を行ってください。
